Class SAGE3Events

A SAGE3 events class to subscribe and publish events across the SAGE3 Backend.

The core purpose of this class is to notify other backend services of events happening on the server. Ex. New User Created, New Board Created, User Moves a Window...etc.

Instead of importing multiple services into sections of the core SAGE3 code to track different events, we publish events when they happen and services subscribe to those events.
